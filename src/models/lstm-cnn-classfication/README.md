## Introduction
This is the implementation for LSTM CNN Text Classfication. <br>
Perform experiments on the English data from [TREC](http://cogcomp.org/Data/QA/QC/)

## Requirement
* python 3.5
* TensorFlow 1.4
* numpy 1.13.1
* tqdm

## Result

> Acc: 88.0%
